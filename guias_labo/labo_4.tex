 \centerline{\bf Laboratorio N$^\circ$ 4: Fourier clásico}
 \bigskip

\bej
Considere la función periódica
\[
f(x)=e^{\sin x},\qquad x\in[0,2\pi].
\]
Trabaje con $n=4,6,8,10$.

\begin{enumerate}[(a)]
\item \textbf{Serie de Fourier e interpolación trigonométrica.}

Para cada $n$:
\begin{itemize}
\item Muestree $f$ en la grilla uniforme $x_j=2\pi j/(2n+1)$, $j=0,\dots,2n$.
\item Calcule numéricamente los coeficientes de Fourier discretos (por DFT/FFT).
\item Construya el interpolante trigonométrico $I_n f$.
\item Evalúe el error de interpolación en el interior del intervalo, por ejemplo sobre una malla fina en $[\delta,2\pi-\delta]$ con $\delta>0$ fijo pequeño.
\item Grafique en una misma figura $f$ e $I_n f$ para $n=4,6,8,10$.
\end{itemize}

Reporte una tabla con $\|f-I_n f\|_\infty$ (en la malla fina interior).

\item \textbf{Cuadratura por trapecios.}

Para cada $n$:
\begin{itemize}
\item Aproxime
\[
I=\int_0^{2\pi}e^{\sin x}\,dx
\]
mediante la regla de trapecios compuesta con $n$ subintervalos:
\[
Q_n=\frac{2\pi}{n}\sum_{j=0}^{n-1}f\!\left(\frac{2\pi j}{n}\right).
\]
\item Tome como referencia $I_{\mathrm{ref}}=Q_{n_{\mathrm{ref}}}$ con $n_{\mathrm{ref}}$ grande (por ejemplo $n_{\mathrm{ref}}=2^{14}$ o mayor).
\item Calcule el error $E_n=|I_{\mathrm{ref}}-Q_n|$ para $n=4,6,8,10$.
\item Grafique la función y el área bajo los trapecios para cada $n$.
\end{itemize}

Comente por qué, aunque los errores locales en cada trapecio son visibles, el error global puede ser de orden mucho mejor para funciones periódicas suaves.
\end{enumerate}
\fej
